iZiSwap-sdk
===========

**iZiSwap-sdk** is a typescript library for interacting with iZiSwap contracts, including 

:core contracts:
   iZiSwapPool, iZiSwapFactory
:periphery contracts:
   LiquidityManager, LimitOrderManager, Swap, Quoter, etc.
